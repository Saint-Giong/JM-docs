### Step 2: For Team Members - Development Setup

<AccordionGroup>
  <Accordion icon="github" title="Clone the Repository">
    Clone the DEVision Job Manager repository:

    ```bash
    git clone https://github.com/saint-giong/devision-job-manager.git
    cd devision-job-manager
    ```
  </Accordion>

  <Accordion icon="code" title="Set Up Development Environment">
    **Frontend Setup:**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```

    **Backend Setup:**
    ```bash
    cd backend
    ./mvnw spring-boot:run
    ```

    See [Development Guide](/development) for detailed setup instructions.
  </Accordion>

  <Accordion icon="docker" title="Run with Docker">
    Start all services with Docker Compose:

    ```bash
    # Copy environment variables
    cp .env.example .env

    # Start services
    docker-compose up -d
    ```

    See [Deployment Guide](/technical/deployment) for full Docker configuration.
  </Accordion>
</AccordionGroup>

## Technology Stack

### Frontend
- React with TanStack Query & Router
- Zustand for state management
- Shadcn/ui + Tailwind CSS
- Headless UI components

### Backend
- Spring Boot microservices
- PostgreSQL with sharding
- Redis for caching
- Kafka for messaging

### Deployment
- Docker containerization
- API Gateway + Service Discovery
- Multi-host deployment
