## Overview

This page showcases the UI mockups for the DEVision Job Manager platform, organized by feature area. These mockups represent the visual design for the Job Manager (JM) subsystem.

## Authentication

### Sign Up

<img src="/images/mockups/signup-1.png" alt="Company signup form with fields for company name, email, password, country, phone number, city, and address" style={{ borderRadius: '0.5rem' }} />

The registration form includes:
- Company Name, Email, and Password (required)
- Password strength requirements displayed
- Country selection with dial code
- Phone number, City, and Address fields
- Google SSO option

### Login

<img src="/images/mockups/login-1.png" alt="Company login form with email and password fields, plus Google SSO option" style={{ borderRadius: '0.5rem' }} />

The login interface provides:
- Email and password authentication
- "Forgot password?" recovery link
- Google SSO alternative

## Dashboard

<img src="/images/mockups/dashboard-1.png" alt="Dashboard overview showing KPI cards, recent job posts, notifications, and draft posts" style={{ borderRadius: '0.5rem' }} />

The dashboard serves as the central hub with:
- **Overview Cards**: Total Job Views and New Applications with trend data
- **Recent Posts**: Quick view of latest job postings
- **Right Panel**: Notifications, Premium features, and Draft posts

## Company Profile

### Profile View

<img src="/images/mockups/profile-1.png" alt="Public company profile page with company info, job posts, and contact details" style={{ borderRadius: '0.5rem' }} />

The public profile displays:
- Company logo, name, and location
- "Who we are looking for" description
- Active job posts with skill tags
- About us and Contact information sidebar

### Profile Edit

<img src="/images/mockups/profile-2.png" alt="Profile editing form with company information and description fields" style={{ borderRadius: '0.5rem' }} />

The edit mode allows updating:
- Company information (name, email, phone, country, city, address)
- Logo upload
- "About us" and "Who we are looking for" descriptions

### Activity Post

<img src="/images/mockups/profile-3.png" alt="Activity post modal for creating company updates" style={{ borderRadius: '0.5rem' }} />

Quick activity posting with emoji, image, and document attachments.

## Job Management

### Jobs List

<img src="/images/mockups/job-1.png" alt="Jobs list view with status filters and job cards showing title, skills, location, and salary" style={{ borderRadius: '0.5rem' }} />

The jobs list features:
- Status filters: All, Drafts, Published, Hiring, Archived
- Grid/List view toggle
- Job cards with status badges, skills, location, and salary
- Applicant count indicators

### Job Detail

<img src="/images/mockups/job-2.png" alt="Job post detail view with full description and applicant count" style={{ borderRadius: '0.5rem' }} />

Job detail view includes:
- Full job description with requirements
- Sidebar with expiry date, location, employment type, salary, and skills
- Tabs for Job Post and Applicants
- Actions: View as Applicants, Share, Edit

### Salary Options

<img src="/images/mockups/job-3.png" alt="Salary input options: Range, About, and Negotiable modes" style={{ borderRadius: '0.5rem' }} />

Three salary input formats:
- **Range**: 30,000 to 80,000
- **About**: ~30,000
- **Negotiable**: Salary to be discussed

## Application Management

### Applicants List

<img src="/images/mockups/application-1.png" alt="Applicants list with filtering by status and skill tags" style={{ borderRadius: '0.5rem' }} />

Applicants are displayed with:
- Status filters: All, Pending, Favorite, Archived, Hiring
- Applicant cards showing name, location, education, email
- Skill tags (Python, Kafka, SQL, etc.)
- Favorite status indicators

### Applicant Detail

<img src="/images/mockups/application-2.png" alt="Applicant detail view with contact info, education, and action buttons" style={{ borderRadius: '0.5rem' }} />

The detail panel shows:
- Contact information (Email, Phone, Location)
- Education and Work experience
- Objective summary
- Action buttons: Favorite, Warning flag, Inbox, Hire
